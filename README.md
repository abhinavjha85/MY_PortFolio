# NeuralRetail - AI Sales Intelligence Platform

## Project Overview
NeuralRetail is an AI-powered sales intelligence platform built for Amdox Technologies. It analyzes retail data to provide demand forecasting, churn prediction, and customer segmentation.

## Live Demo
Live Dashboard: https://myportfolio-hoqgbmv9udk7dxj7wrfbef.streamlit.app

## Key Results
- Total Revenue Analyzed: £8,911,408
- Total Orders: 18,536
- Total Customers: 4,339
- Countries: 37
- Churn Model AUC: 1.0

## Tech Stack
- Python 3.12
- Pandas, NumPy
- Prophet (Demand Forecasting)
- XGBoost (Churn Prediction)
- KMeans (Customer Segmentation)
- Streamlit (Dashboard)
- Plotly (Charts)

## How to Run
1. Clone the repo
2. pip install -r requirements.txt
3. streamlit run dashboard/app.py
